# Pixabay API

Aplicación web que permite buscar imágenes en la **API de Pixabay**, desarrollada con **HTML, JavaScript y Sass**.  
El objetivo es practicar el consumo de APIs REST, manejo de asincronía y construcción de una galería dinámica con diseño moderno y responsivo.

---

## Características
- Barra de búsqueda para consultar imágenes en Pixabay.
- Renderizado dinámico de resultados en una **galería responsiva**.
- Estilos modernos con **Sass** compilado a CSS.
- Diseño **responsive** para desktop y mobile.

---

## Tecnologías utilizadas
- **HTML5**  
- **CSS3 / Sass (variables, anidación, modularización)**  
- **JavaScript (fetch, asincronía, DOM dinámico)**  
- **Pixabay API**
